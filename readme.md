## About Tax Calculator

This is project of Tax Calculator Challenge, contains:
1. API (token and calculate)
2. Docs of API
3. Docker file

The project is built by PHP using Laravel framework.

To start project:

1. php artisan migrate
2. php artisan db:seed --class=ItemsTableSeeder
3. php artisan serve
